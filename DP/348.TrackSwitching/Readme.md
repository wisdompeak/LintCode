### 348. TrackSwitching

#### Description
```
在一场赛车比赛中，一共有三条赛道，初始时你在2号车道。赛车可以在遇到障碍时切换赛道。
在赛道上有一些障碍，它们的位置 obstacles 从起点出发由近到远地列出，obstacles_i 代表第 i 个障碍在哪条赛道上。
障碍在赛道上是分散的，即不会同时有两个障碍挡住赛车的去路。
请计算出到达终点最少换几次赛道。

赛道上的障碍数量为 n, 1≤n≤5*10^4。
赛道的编号分别是 1,2,3。

1 3 2 。。。
|—|—|—|—|—|
```

#### 分析：
```
本轮在各赛道需要的切换次数，完全取决于前一轮在各赛道上的切换次数。典型的DP。
令dp[i][j]表示在第i轮处于第j赛道，所需要的最少切换次数。注意，我们认为障碍物obstacle[i]位于第i轮和第i+1轮之间。

状态转移方程可以有多种多样。这里提供一个dp[i][j]往dp[i+1][j]转移的方法。其中i表示轮次，j表示赛道。

令obs表示当前第i轮次的障碍物位置。

    i      				i+1
1  dp[i][1]  --------> 	 |		
2  dp[i][2]  -------->  ||
3  dp[i][3]    X		||

1. 对于j!=obs的这些赛道，继续往前冲毫无疑问的最优方案。即 dp[i+1][j] = dp[i][j]
2. 对于有障碍物的赛道，抵达dp[i+1][obj]的策略，只能是先从其他赛道到达i+1，再切换一次赛道
	dp[i+1][3] = min(dp[i][1]+1, dp[i][2]+1)

最终答案是 dp[n-1][j]最小的第一个。（注意，虽然最后一轮之后还有一个障碍，但已经不影响答案了）
```
