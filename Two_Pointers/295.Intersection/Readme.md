### 295. Intersection

#### Description
```
给定两个排序后的区间序列，序列内每个区间两两互不相交，
返回两个序列相交的区间的下标

1<=len(a),len(b)<=1e5
abs(max(a))<=1e9

Example
输入: a = [[0,3], [7,10]] , b = [[-1,1],[2,8]]
输出: ans = [[0,0],[0,1],[1,1]]

        _____   ______

  ______   _______
```

#### 思路
```
我们永远只需要考虑当前a的第一个区间和当前b的第一个区间的交叠关系。

1. 如果两者完全不相交，下一步我们一定会忽略掉整体靠前的那个区间。因为该区间不可能与另一组的任何区间相交。
_____   ....

      ______  ...

2. 如果两者相交，我们就记录下这一对，然后忽略掉最先结束的那个区间。因为该区间不可能与另一组的任何其他区间再相交。
Ex1:
______   ....

   ______  ...

Ex2:
_____________   ....

   ______  ...

两组区间的指针依次递增直至结束。
```
